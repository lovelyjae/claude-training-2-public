# 🅱️ Claude Code 2차 실습 워크북 · 트랙 B
## 대시보드 자동화 — 엑셀을 한눈에 보이게

> **오늘의 목표:** 내 반복업무(데이터 정리)를 `claude` **한 줄(또는 예약)** 로 대시보드(`dashboard.html`)로 만들고, 동료 앞에서 시연합니다.
> **대상:** 1차에서 Claude Code 설치를 마친 분 · **소요:** 4~5시간
> 모든 STEP은 **① 복붙 블록 → ② 이렇게 나오면 성공 → ③ ✋체크포인트 → 🆘안 될 때** 순서입니다.
> 이 워크북은 **트랙 B 전용**입니다. (Notion 보고를 만들 분은 트랙 A 워크북으로)

# STEP 0 · 내 자동화 후보 찾기  ⏱ 10분

매번 다시 시키는 일 = 자동화 후보입니다. 1차 때 쌓인 내 대화 기록에서 직접 찾아봅니다.

**📋 복붙 블록 0-1** — Claude Code에 그대로 붙여넣기
```
내 ~/.claude/projects 폴더의 최근 대화 기록을 분석해서,
내가 비슷한 요청을 반복한 작업 Top 5를 찾아줘.
각각 ⓐ무슨 일을 ⓑ몇 번쯤 시켰는지 ⓒ자동화하면 절약될 시간을 표로 정리하고,
그중 '집계·현황판·시각화'처럼 대시보드로 만들기 좋은 일 1개를 추천해줘.
```

**② 이렇게 나오면 성공**
- "반복한 작업 Top 5" 표가 나온다. (기록이 적으면 적게 나와도 OK)

> 📸 *[스크린샷 자리: Top 5 표 결과]*

**🆘 안 될 때**
- 기록이 거의 없다 → 괜찮아요. STEP 1 설문이 후보를 대신 잡아줍니다.

(트랙 B가 추천되면 그대로, 다른 트랙이 나와도 오늘은 **B로 진행**합니다.)

> 📸 *[스크린샷 자리: 설문 결과 화면 — 기획서 표]*

설문 결과를 보고 아래 칸을 채우세요. (캡처만 해도 됩니다)

**내 자동화 한 줄 정의:**

# STEP 2 · 공통 레일 만들기  ⏱ 45분  ★게이트 A

> 모든 자동화는 같은 골격입니다: **① 규칙 적기(CLAUDE.md) → ② 한 줄 실행 → ③ 예약(STEP 3)**

### B-1. 규칙 만들기
`2차실습/트랙B` 폴더(엑셀 파일 포함)를 Claude Code로 연 뒤, `CLAUDE.md`를 만들고 붙여넣습니다.

**📋 복붙 블록 B-1**
```markdown
# 대시보드 자동화 규칙
## WHAT: 데이터를 한눈에 보는 대시보드를 만든다
## 입력: ./데이터.xlsx
## 출력: dashboard.html (핵심지표 카드 + 표 + 막대그래프)
## 규칙: 숫자는 천단위 콤마, 최신 데이터 기준으로 갱신
## 검증: 끝나면 핵심지표 3개를 요약해서 보여줘
```

### B-2. 한 줄로 실행
**📋 복붙 블록 B-2**
```
claude "데이터.xlsx로 대시보드 만들어줘"
```

**② 이렇게 나오면 성공**
- `dashboard.html`이 생기고, 브라우저로 열면 표·차트·핵심지표가 보인다.

> 📸 *[스크린샷 자리: 생성된 dashboard.html 화면]*

### ✋ ★게이트 A — 전원 통과 지점
- [ ] **"한 줄 실행"으로 `dashboard.html`이 한 번 만들어졌다**

> 🎉 이 순간 **자동화 1개(Lv2) 완성** = 졸업 조건 충족. 안 되면 손 들기! 🙋

**🆘 안 될 때**
| 증상 | 해결 |
|---|---|
| 파일을 못 찾는다 | 파일 이름이 `데이터.xlsx` 와 같은지 / 트랙B 폴더에서 열었는지 확인 |
| 차트가 안 보인다 | "막대그래프를 Chart.js로 그려줘" 처럼 그리는 방법을 지정 |
| 숫자가 깨져 보인다 | CLAUDE.md `## 규칙` 에 "천단위 콤마·소수점 1자리" 명시 |
| 결과가 이상하다 | CLAUDE.md `## 출력` 을 더 구체적으로 (어떤 카드·어떤 표) |

# STEP 5 · 견고화 (안 깨지게)  ⏱ 35분

### 5-1. "왜 안 돼!"를 줄이는 3원칙
1. **"이거" 금지** — 대상을 1개 명시. ("이거 또 만들어줘" ❌ → "데이터.xlsx로 대시보드 만들어줘" ✅)
2. **한 번에 한 의도** — 지시+질문+변심을 한 줄에 섞지 않기.
3. **항상 검증** — CLAUDE.md에 `## 검증: 끝나면 핵심지표 3개를 요약해서 보여줘`를 넣기.

### 5-2. 안전장치 체크리스트
- [ ] 민감정보(고객명/개인정보/단가 등)는 엑셀에서 뺐거나 가렸다
- [ ] 결과 검증 한 줄을 CLAUDE.md에 넣었다
- [ ] 예약 자동화는 **첫 2~3회 대시보드 숫자를 눈으로 확인**하기로 했다

### 5-3. 자동화가 멈췄을 때 — 자가진단 3줄
1. **입력이 그대로 있나?** (`데이터.xlsx` 이름·위치·시트 구조 변경 여부)
2. **데이터 형식이 바뀌었나?** (열 이름·날짜 형식이 달라지면 다시 알려주기)
3. **그냥 물어보기** → `claude "이 자동화 왜 안 됐는지 봐줘"`
