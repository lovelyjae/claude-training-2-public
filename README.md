# Claude Code 비개발자 교육 2차 · 내 손으로 만드는 자동화 1개

1차(설치~맛보기) 수료생 대상 중급 과정. **4~5시간 실습으로 자동화 1개를 반드시 완성**합니다.

## 구성
| 파일 | 용도 |
|---|---|
| `index.html` | 강의안 (슬라이드쇼) — 키보드 `←→` 넘김, `F` 전체화면, `O` 전체보기, `S` 발표자 노트, `P` 인쇄 |
| `survey.html` | 자동화 기획 진단 설문 — 6문항 → 추천 트랙 + 기획서 초안 |
| `workbook.md` | 실습 워크북 (STEP 0~6) |
| `2차실습/트랙A/이번주_메모.md` | 트랙 A(Notion) 실습용 샘플 |
| `2차실습/트랙B/데이터.xlsx` | 트랙 B(대시보드) 실습용 샘플 |

## 흐름
각성 → **기획(설문)** → 공통 레일(Notion/대시보드 분기) → 무인화(예약) → 내 업무 치환 → 견고화 → 졸업
